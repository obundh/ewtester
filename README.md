# 측정 장비 시뮬레이터

집에서 전자파 시험 흐름과 보조 계측기 조작을 함께 연습할 수 있도록 만든 정적 웹 기반 GUI 시뮬레이터입니다.

## 실행

브라우저에서 `index.html`을 바로 열면 됩니다.

로컬 서버를 쓰고 싶다면:

```powershell
python -m http.server 8000
```

그 뒤 `http://localhost:8000`으로 접속하면 됩니다.

## 현재 구성

- 홈 화면에서 차폐 실험 / PCI 실험 구분
- 오실로스코프 시뮬레이터
- 스펙트럼 분석기 시뮬레이터
- 시그널 제네레이터 시뮬레이터
- 한국어 / English 전환 버튼
- 앱 내부 사용설명서 화면
- 데이터 분리 파일 `sim-data.js`

## 스펙트럼 분석기 기능

- LOS 송신 / 수신 안테나 선택
- 안테나 대역 인식 예시: `바이코니컬 안테나 [20 MHz ~ 200 MHz]`
- 송신 주파수, 출력, 거리 설정
- 기준값 저장 후 현재 트레이스와 오버레이 비교
- 수신 안테나 차폐 상자 시뮬레이션
- 스테이지에서 `송신 / 수신 / 차폐함` 슬롯으로 안테나 드래그 배치
- 차폐 재질, 차폐 성능, 틈/개구 누설량 설정
- 피크 주파수, 피크 레벨, 기준 대비 delta, 차폐 감쇠, 경로 손실, 노이즈 플로어 표시

## 시그널 제네레이터 기능

- 주파수 값 + 단위 `Hz / kHz / MHz / GHz` 설정
- `LEVEL` 버튼 기반 dBm 레벨 조정
- RF 출력 ON/OFF
- 차폐 실험 송신 주파수 / 송신 출력과 직접 연동
