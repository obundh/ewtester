# 측정 장비 시뮬레이터 사용설명서

## 개요

이 프로젝트는 집에서 전자파 시험 흐름과 계측기 조작 감각을 함께 익히기 위한 연습용 GUI 시뮬레이터입니다.
현재 홈 화면은 차폐 실험과 PCI 실험 중심으로 구성되어 있고, 오실로스코프/스펙트럼 분석기/시그널 제네레이터는 보조 도구로 사용할 수 있습니다.

## 시작 방법

1. `index.html`을 브라우저에서 엽니다.
2. 홈 화면에서 `차폐 실험` 버튼으로 먼저 진입합니다. `PCI 실험`은 현재 준비 중입니다.
3. 상단 우측의 언어 버튼으로 `한국어`와 `English`를 전환할 수 있습니다.
4. 상단 메뉴에서 `시그널 제네레이터`로 들어가 송신 주파수와 레벨을 먼저 맞춘 뒤 차폐 실험으로 돌아올 수 있습니다.

## 스펙트럼 분석기 사용 흐름

1. 송신 안테나와 수신 안테나를 선택합니다.
2. 예를 들어 `바이코니컬 안테나 [20 MHz ~ 200 MHz]`를 선택하면 해당 대역이 analyzer 환경에 반영됩니다.
3. 송신 주파수, 송신 출력, LOS 거리를 맞춥니다.
4. 차폐 상자를 끄고 `기준값 저장`을 누릅니다.
5. 수신 안테나를 차폐 상자 안에 둔 상태를 가정해 차폐 상자를 켭니다.
6. 차폐 재질, 차폐 성능, 누설량을 바꾸며 기준값 대비 delta를 비교합니다.

## 시그널 제네레이터 사용 흐름

1. `시그널 제네레이터` 화면에서 주파수 값을 입력합니다.
2. `Hz / kHz / MHz / GHz` 단위를 선택합니다.
3. `LEVEL` 버튼을 누른 뒤 dBm 값을 조정합니다.
4. RF 출력 ON/OFF를 바꾸면 analyzer 수신 상태에도 반영됩니다.

## Analyzer 화면 설명

### 왼쪽 패널

- 송신 안테나 / 수신 안테나 선택
- 송신 소스 프로파일 선택
- 송신 주파수, 출력, 거리 설정
- 차폐 상자 재질, 성능, 누설량 설정
- center / span / RBW 설정

### 가운데 화면

- 현재 스펙트럼 트레이스 표시
- 기준값 저장 후 reference 트레이스 오버레이 표시
- 공통 대역과 sweep 범위 확인

### 오른쪽 패널

- 피크 주파수
- 피크 레벨
- 기준 대비 delta
- 차폐 감쇠
- 경로 손실
- 노이즈 플로어
- 안테나 공통 대역 상태와 차폐 상태 요약

## 참고

- 이 시뮬레이터는 학습용이므로 실제 장비의 모든 세부 메뉴를 완전히 재현하지는 않습니다.
- 다음 단계에서는 PCI 실험과 차폐 실험 preset 저장 기능을 붙이면 흐름이 더 완성됩니다.
